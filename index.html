Modelos de Cores: RGB e CMYK

Modelo RGB (Red, Green, Blue)

O modelo RGB é um modelo de cores aditivo usado em dispositivos como monitores de computador, TVs, câmeras digitais, e smartphones. A ideia central do RGB é que todas as cores podem ser criadas através da combinação de diferentes intensidades de luz vermelha (Red), verde (Green) e azul (Blue). Ao combinar essas três cores em diferentes intensidades, é possível gerar uma vasta gama de cores.

Exemplo de como o modelo funciona:

  • Vermelho puro: 255, 0, 0 (máxima intensidade de vermelho, sem verde e azul).
  • Branco: 255, 255, 255 (máxima intensidade de todas as cores).
  • Preto: 0, 0, 0 (sem luz emitida).

As cores mais próximas do branco são obtidas quando todas as três cores são combinadas na sua intensidade máxima (255 para cada cor), enquanto a cor preta é obtida quando nenhuma luz é emitida (0 para todas as cores).

Funcionamento do Pixel no RGB

Um pixel em uma tela RGB é formado por três subpixels: um vermelho, um verde e um azul. Cada subpixel emite uma quantidade específica de luz de sua cor correspondente. O olho humano percebe a combinação dessas luzes como uma única cor. O pixel pode ser ajustado para variar as intensidades de cada subpixel (geralmente de 0 a 255), criando diferentes tons.

Exemplo de ajustes nos subpixels:

  • Vermelho puro: 255, 0, 0
  • Branco: 255, 255, 255
  • Preto: 0, 0, 0

Modelo CMY (Cyan, Magenta, Yellow)

O modelo CMY é um modelo de cores subtrativo utilizado em impressão, onde as cores são criadas pela subtração de luz. Ele usa três cores primárias: ciano (Cyan), magenta (Magenta) e amarelo (Yellow). Quando combinadas, elas absorvem luz de diferentes partes do espectro visível, resultando em cores.

Exemplo de combinações do modelo CMY:

  • Ciano + Magenta: Produz azul.
  • Magenta + Amarelo: Produz vermelho.
  • Ciano + Amarelo: Produz verde.

A mistura das três cores primárias pode resultar em um tom próximo ao preto, mas para melhorar a qualidade das impressões, é adicionado o preto (K), formando o modelo CMYK.

Comparação entre RGB e CMYK

Enquanto o modelo RGB é utilizado em dispositivos que emitem luz, como monitores de computador e TVs, o modelo CMYK é mais comum em impressão. As principais diferenças são:

  • RGB: A adição de cores (luz) gera uma gama maior de cores. Cada componente (R, G, B) varia de 0 a 255.
  • CMYK: A adição de mais tinta (C, M, Y) resulta em uma cor mais escura. O K (preto) é adicionado para melhorar a profundidade das cores.

Resumindo:

  • RGB é mais utilizado em telas.
  • CMYK é utilizado para impressão.
  • RGB mistura luzes para gerar cores, enquanto CMYK usa pigmentos para subtrair luz e criar cores.
